
## filterx

A fast command-line tool to filter lines by column-based expression.

## Features
- 🚀 Filter lines by column-based expression
- 🎨 Support multiple input formats e.g. vcf/sam/fasta/fastq/gff/bed/csv/tsv
- 🎉 Cross-platform support
- 📦 Easy to install
- 📚 Rich documentations
